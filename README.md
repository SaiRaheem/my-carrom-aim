# My Carrom Aim App

This is a custom Carrom Aim assistant app built for low-latency and precision.

## How to build (Option 2: GitHub Actions)

1. Create a **Private** Repository on GitHub.
2. Upload all the files inside the `MyCarromAimApp` folder to that repository.
3. Every time you push a change to the `main` branch, GitHub will automatically build your APK.
4. Go to the **"Actions"** tab on your GitHub repository.
5. Click on the latest run, and under **"Artifacts"**, you will find the download link for `carrom-aim-app-debug`.

## Current Features
- Transparent Overlay Service
- Permission handling for Android 6.0+
- Foreground service for background persistence
